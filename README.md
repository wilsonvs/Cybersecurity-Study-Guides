# Cybersecurity Study Guides

A central learning hub for cybersecurity certification study notes. This repository keeps the main study paths in one place so learners can quickly access focused notes for CompTIA Security+, Google Cybersecurity Professional Certificate, and ISC2 Certified in Cybersecurity.

Each study guide is maintained as a separate repository so the notes stay organized, searchable, and easy to update.

## Recommended Study Path

1. Start with [ISC2 Certified in Cybersecurity CC](https://github.com/wilsonvs/ISC2-CC-Study-Guide) for core cybersecurity fundamentals.
2. Continue with [Google Cybersecurity Professional Certificate](https://github.com/wilsonvs/Google-Cybersecurity) for practical SOC, tools, Linux, SQL, Python, and detection concepts.
3. Use [CompTIA Security+ SY0-701](https://github.com/wilsonvs/CompTIA-Security-SY0-701) for broader exam preparation and deeper security coverage.

## What These Notes Are Designed For

## How to Use This Hub

1. Pick the certification or learning path you are currently studying.
2. Open the dedicated repository from the index above.
3. Study one module, domain, or section at a time.
4. Review important terms and quick-check questions.
5. Use the review checklists before moving to the next section.
6. Return to this hub when switching between certifications.
