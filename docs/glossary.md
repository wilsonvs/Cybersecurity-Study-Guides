# Cybersecurity Glossary

A quick reference for common cybersecurity terms used across the study guides.

## Core Terms

- Asset - Anything valuable that needs protection.
- Threat - A possible cause of harm.
- Vulnerability - A weakness that can be exploited.
- Risk - The chance that a threat exploits a vulnerability and causes impact.
- Control - A safeguard used to reduce risk.
- Incident - A security event that may harm systems, data, or operations.
- IOC - Indicator of Compromise; evidence that may show malicious activity.
- SIEM - Security Information and Event Management platform used to collect and analyze logs.
- SOC - Security Operations Center responsible for monitoring and response.

## Review Tip

For each term, practice explaining what it means, why it matters, and one example of where it appears in security work.
