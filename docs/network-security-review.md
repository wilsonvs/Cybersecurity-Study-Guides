# Network Security Review Notes

## Review Areas

- Exposed services
- Firewall rules
- Remote access paths
- VPN configuration
- DNS activity
- Unusual outbound connections
- Network segmentation
- Wireless security

## Questions To Ask

- Which services are reachable from the internet?
- Are administrative ports restricted?
- Are firewall rules documented?
- Is remote access protected with MFA?
- Are critical systems segmented?

## Evidence To Collect

- Firewall logs
- DNS logs
- VPN logs
- Network diagrams
- Asset inventory
- Change records
