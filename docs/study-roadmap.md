# Cybersecurity Study Roadmap

Use this roadmap to move through the study guides in a practical order.

## 1. Build Core Foundations

Start with the ISC2 Certified in Cybersecurity CC notes.

Focus areas:

- Security principles
- Business continuity, disaster recovery, and incident response
- Access control
- Network security
- Security operations

Recommended repo: https://github.com/wilsonvs/ISC2-CC-Study-Guide

## 2. Connect Concepts With Practical Skills

Continue with the Google Cybersecurity Professional Certificate notes.

Focus areas:

- SOC monitoring basics
- Risk management
- Network security concepts
- Linux and SQL
- Assets, threats, and vulnerabilities
- Detection and response
- Python automation

Recommended repo: https://github.com/wilsonvs/Google-Cybersecurity

## 3. Deepen Exam Coverage

Use the CompTIA Security+ SY0-701 notes for broader coverage and repeated review.

Focus areas:

- Threats, attacks, and vulnerabilities
- Security architecture
- Security operations
- Governance, risk, and compliance
- Identity and access management
- Incident response

Recommended repo: https://github.com/wilsonvs/CompTIA-Security-SY0-701

## Weekly Study Method

1. Study one section or module at a time.
2. Write down important terms in your own words.
3. Complete the practice tasks.
4. Answer the quick-check questions without looking first.
5. Revisit weak topics before moving ahead.

## Review Method

Use each repository's checklist before final review. The goal is not only memorization, but being able to explain what a concept means, why it matters, and how it appears in practical cybersecurity work.
