# SIEM Query Study Plan

## Goal

Build comfort with asking security questions using logs.

## Practice Questions

- Which users had repeated failed logins?
- Which source IPs generated the most blocked firewall events?
- Which accounts logged in outside normal hours?
- Which hosts connected to rare external domains?
- Which admin actions happened in the last 24 hours?

## Study Pattern

1. Define the question.
2. Identify the log source.
3. Select useful fields.
4. Filter the noise.
5. Summarize the result.
6. Document what the result means.
