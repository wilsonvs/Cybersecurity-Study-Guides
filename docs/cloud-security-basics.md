# Cloud Security Basics

## Core Concepts

- Shared responsibility model
- Identity and access management
- Least privilege
- Network security groups
- Logging and monitoring
- Encryption
- Backup and recovery
- Secure configuration

## Review Questions

- Who is responsible for the cloud platform?
- Who is responsible for data and configuration?
- Are admin accounts protected with MFA?
- Are logs enabled?
- Are storage resources public or private?
- Are secrets stored safely?

## Study Tip

Cloud security starts with identity, logging, configuration, and data exposure. Review those areas first.
