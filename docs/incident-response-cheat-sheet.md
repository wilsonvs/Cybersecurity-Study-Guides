# Incident Response Cheat Sheet

## Basic Flow

1. Prepare tools, access, contacts, and playbooks.
2. Detect suspicious activity.
3. Analyze evidence and confirm scope.
4. Contain the impact.
5. Eradicate the cause.
6. Recover safely.
7. Document lessons learned.

## Triage Questions

- What triggered the alert?
- Which user, host, IP, or application is involved?
- When did it happen?
- Is the asset critical?
- Is there supporting evidence?
- Does this require escalation?

## Documentation Fields

- Timestamp
- Source of evidence
- Affected asset
- Observed activity
- Action taken
- Next step
