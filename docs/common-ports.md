# Common Ports Quick Reference

| Port | Protocol | Use |
| --- | --- | --- |
| 22 | SSH | Secure remote administration |
| 25 | SMTP | Email sending |
| 53 | DNS | Name resolution |
| 80 | HTTP | Web traffic |
| 110 | POP3 | Email retrieval |
| 143 | IMAP | Email retrieval |
| 443 | HTTPS | Secure web traffic |
| 3389 | RDP | Windows remote desktop |

## Study Method

When reviewing ports, connect each port with a security question. Example: if RDP is exposed to the internet, what risk does that create and what controls should reduce it?
