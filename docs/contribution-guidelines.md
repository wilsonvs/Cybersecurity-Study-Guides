# Contribution Guidelines

This repository is focused on clean, beginner-friendly cybersecurity study material.

## Good Contributions

- Fix unclear explanations.
- Add missing definitions.
- Improve formatting.
- Add useful review questions.
- Update outdated links.
- Add practical examples.

## Style

- Use simple language.
- Keep sections easy to scan.
- Avoid unnecessary jargon.
- Do not copy official course or exam content.
- Cite official sources where useful.

## Structure

Keep files organized by topic. Prefer one clear file over many small nested files when the content is easier to study in one place.
