# Monthly Maintenance Plan

Use this checklist to keep the study repositories current.

## Monthly Review

- Check official certification pages for outline changes.
- Verify external links.
- Review README descriptions.
- Clean outdated wording.
- Add missing glossary terms.
- Improve sections that are too brief.
- Check repo topics and descriptions.

## Quarterly Review

- Review all study paths.
- Add new practical examples.
- Update threat research references.
- Archive duplicate or unused files.
- Confirm pinned repositories still match the profile goal.
