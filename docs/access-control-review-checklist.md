# Access Control Review Checklist

## Account Review

- Confirm active users.
- Remove stale accounts.
- Disable orphaned accounts.
- Review privileged accounts.

## Permission Review

- Check least privilege.
- Review group membership.
- Remove unnecessary access.
- Validate role-based permissions.

## Authentication Review

- Confirm MFA coverage.
- Check password policy.
- Review failed login trends.
- Monitor risky sign-ins.

## Documentation

Record reviewer, date, systems checked, issues found, and actions taken.
